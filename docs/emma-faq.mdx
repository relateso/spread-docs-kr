## 자동화 캠페인을 중단하려면 어떻게 하나요?
우측 상단의 중지 버튼을 클릭하면 자동화가 즉시 중단됩니다.
중단 이후에는 새로운 리드가 추가되어도 이메일이 발송되지 않습니다.


<br/>
## 수신 거부를 하면 이후 모든 자동화에서 완전히 제외되나요?
네. 수신 거부는 모든 자동화보다 우선합니다.

리드가 수신 거부를 하면:  
- 모든 자동 이메일 발송이 즉시 중단되고  
- 이후 어떤 자동화 흐름에도 다시 포함되지 않습니다.  

이는 법적·운영적 안정성을 위한 기본 원칙이며, Emma는 이 상태를 자동으로 관리합니다.


<br/>
## 이미 발송한 캠페인에도 자동화를 연결할 수 있나요?
네, 가능합니다.
이미 발송한 캠페인에서 열람, 클릭 등 이메일 반응을 조건으로
리드를 새로운 대상 리스트로 이동시키고,
그 리스트를 시작점으로 자동화 캠페인이나 시퀀스를 연결할 수 있습니다.


<br/>
## 자동화 캠페인과 시퀀스는 어떻게 연결되나요?
자동화 캠페인의 대상 리스트를
다음 자동화 캠페인 또는 시퀀스의 시작 리스트로 설정할 수 있습니다.
이를 통해 이메일 반응 → 추가 안내 → 세일즈 팔로업까지 자연스럽게 연결할 수 있습니다.


<br/>
## 테스트 발송은 실제 리드에게도 전송되나요?
아니요.
테스트 발송은 워크스페이스 내부 사용자 중 선택한 이메일 주소로만 발송됩니다.
실제 리드에게 발송되기 전, 내용과 레이아웃을 미리 확인하는 용도로 사용하세요.
