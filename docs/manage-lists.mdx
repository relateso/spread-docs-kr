## Overview

이 문서는 **구독자 리스트를 안전하게 관리하는 방법**을 다룹니다. 여기서 '안전하게'란, 유효하지 않은 주소로 인한 바운스를 줄여 **이메일 전달률(deliverability)과 도메인 평판을 지키는 것**을 의미합니다.

특히 수천~수만 명 규모의 리스트를 가지고 있고 한동안(몇 개월 이상) 이메일을 보내지 않았다면, 본격적인 발송 전에 **웜업(warm-up)과 리스트 클리닝**을 먼저 진행하는 것이 좋습니다.

<Warning>
  임시·가짜 이메일 주소로 발송하면 Soft/Hard 바운스가 발생합니다. 바운스가 누적되면 이후 발송이 제한되고, 계정이 차단되거나 도메인 평판을 회복하기 어려워질 수 있습니다. 한 번 손상된 평판은 복구에 오랜 시간이 걸리거나 사실상 불가능한 경우도 있으므로, 처음부터 깨끗한 리스트를 유지하는 것이 중요합니다.
</Warning>

## 구독자를 추가할 때 지켜야 할 것

이메일 주소(구독자)를 스프레드에 추가할 때는 다음을 지켜주세요.

1. **사전에 이메일 수신 동의를 받은 구독자만** 추가합니다.
2. 수신 동의 Form에 **reCAPTCHA, Botpoison** 등 봇 차단 장치를 추가해 가짜 가입을 막습니다.
3. (가능하다면) **Double Opt-In** 절차로 수신 동의를 한 번 더 확인합니다.
4. **수신거부했거나 바운스 이력이 있는 주소**로는 다시 발송하지 않습니다.

## 스프레드의 자동 보호 기능

스프레드는 깨끗한 리스트를 유지할 수 있도록 다음을 자동으로 처리합니다.

### 바운스·구독 상태 자동 관리

스프레드를 통해 모든 이메일(트랜잭션, 마케팅, 제품 업데이트)을 발송한다면, **바운스·수신거부 관리를 따로 하지 않아도 됩니다.** 스프레드가 각 구독자의 구독 상태(바운스 여부, 수신거부 여부)를 자동으로 추적해, 발송 가능한 대상에게만 이메일을 보냅니다.

또한 캠페인 발송 중 **발송 1,000건당 바운스 발생률이 5%를 넘으면 자동으로 캠페인을 중단**하고 안내드려 추가 피해를 줄입니다.

### 바운스 가능성이 높은 주소 자동 감지

스프레드는 등록되는 모든 구독자를 대상으로 자동 검사를 실행해 **바운스 가능성이 높은 이메일 주소를 사전에 감지**합니다. 다만 이는 가능성을 예측하는 것이며 100% 정확하지는 않으므로, 바운스를 완전히 예방하지는 못합니다.

## 리스트 만들기

리스트는 이메일 캠페인을 보낼 **구독자 목록**을 의미합니다. 리스트를 만들어 이메일 수신자로 지정하는 방법은 [리스트 만들고 수신자 지정하기](/lists-and-recipients)에서 자세히 안내합니다.
