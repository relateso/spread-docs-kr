## Overview
Spread에서 발송된 캠페인 또는 자동화 캠페인을 웹 링크로 발행하여 외부에 공유할 수 있습니다. 이메일을 실제로 받지 않은 사람도, 링크만으로 동일한 내용을 브라우저에서 확인할 수 있습니다.
슬랙, 메신저, 노션, 문서, 고객 전달용 자료 등 다양한 용도로 활용해 보세요.

## 공유 링크 생성 조건
- **발송 완료된 캠페인/자동화 캠페인만** 웹 링크 생성이 가능합니다.
- 초안 상태(작성 중)에는 `발행하기` 버튼이 표시되지 않습니다. 즉, 발송 후에만 웹 링크를 생성할 수 있습니다.
- 초안 상태에서 공유하고 싶은 경우에는, 다음 방법으로 공유할 수 있습니다:
    - 브라우저 상단 주소창의 페이지 URL을 그대로 복사하여 전달
    - 해당 워크스페이스 접근 권한이 있는 팀원만 열람 가능
    - 외부 사용자에게는 초안 상태 공유 불가

## 향후 업데이트 예정

향후에는 발행된 이메일을 한곳에 모아 볼 수 있는 페이지가 제공될 예정이며, 해당 페이지에서 방문자가 이메일을 입력하면 구독 리스트에 바로 추가할 수 있는 기능도 지원할 계획입니다.
