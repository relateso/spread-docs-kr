## Spread | Emma 자동화의 기본 세트

Emma의 모든 자동화는 다음과 같은 기본 세트를 반복하며 동작합니다.  
 
<div style={{ display: "flex", justifyContent: "center" }}>
  <span style={{ display: "inline-block", border: "1px solid #e5e7eb", padding: "2px 8px", borderRadius: "6px" }}><strong>리스트 → 이메일 → 리스트</strong></span>
</div>


하나의 자동화 세트는 다음 세 가지 요소로 구성됩니다.  
	1.	**시작 리스트** : 이메일을 발송할 대상 리드가 모여 있는 리스트
	2.	**이메일**: 캠페인, 자동화 캠페인 또는 시퀀스
	3.	**대상 리스트**: 이메일 반응에 따라 리드가 이동하는 다음 리스트
<span id="emma-campaign" style={{ scrollMarginTop: "96px" }}></span>
이 구조에서 리스트는 항상 이메일의 앞과 뒤에 위치하며, Emma는 이 세트를 연결해 확장하는 방식으로 설계됩니다.
이를 통해 같은 이메일을 받았더라도 리드의 반응에 따라 서로 다른 다음 흐름으로 이어집니다.

<br />
## 자동화 세트를 구성하는 세 가지 요소
위 자동화 세트를 구성하는 리스트, 캠페인, 시퀀스의 역할을 살펴보겠습니다.

>### 리스트 (List)
리스트는 연락처와 조직을 활용해 만드는 일련의 고객 목록입니다. Emma에서는 자동화의 시작점이자, 다음 단계로 넘어가기 위한 연결 지점입니다. 
- 모든 리드는 반드시 하나 이상의 리스트에 속해 있습니다.  
- 모든 이메일 발송은 **시작 리스트**를 기준으로 이루어집니다.  
- 사전에 설정한 조건을 만족하는 리드가 **대상 리스트**로 자동 이동합니다.


>### 자동화 캠페인 (Triggered Campaigns)

캠페인은 리스트에 속한 리드에게 한 번에 발송되는 이메일입니다.
주로 동일한 메시지를 다수의 리드에게 전달할 때 사용됩니다.   

- 캠페인의 사용 목적 : 뉴스레터, 너처링 콘텐츠, 마케팅 공지 등

시작 리스트를 기준으로 발송되며, 발송 이후에는 아래와 같은 **리드의 이메일 반응**을 기준으로 새로운 리스트를 자동으로 생성할 수 있습니다.

 
<div style={{ paddingLeft: "16px" }}>
  <div>✔️ <strong>이메일 오픈 여부</strong></div>
  <div>✔️ <strong>링크 클릭 여부</strong></div>
  <div>✔️ <strong>수신 거부 여부</strong></div>
</div>


<Note>
    자동화 캠페인 vs 캠페인
</Note>
<span id="emma-sequence" style={{ scrollMarginTop: "96px" }}></span>
>### 시퀀스 (Sequence)

시퀀스는 리드와의 대화를 이어가는 개별 이메일 흐름입니다.
주로 리드의 반응을 전제로 후속 커뮤니케이션을 이어갈 때 사용됩니다.

- 시퀀스의 사용 목적 : 세일즈 팔로업, 미팅 제안, 응답 유도 이메일 등 


시퀀스에서는 **리드의 회신 여부**를 기준으로 리드를 다시 분류할 수 있습니다.   

<div style={{ paddingLeft: "16px" }}>
  <div>✔️ <strong>회신</strong></div>
  <div>✔️ <strong>미회신</strong> : 미회신 기준은 기본적으로 7일로 설정되어 있으며, 각 시퀀스 설정 페이지에서 팀에 맞게 변경할 수 있습니다.</div>
</div>

## 자동화 흐름 설정하기
#### Step 1. 자동으로 발송할 이메일을 작성합니다  
캠페인, 자동화 캠페인 또는 시퀀스 페이지의 상단의 `작성` 탭에서 자동으로 발송할 이메일 내용을 먼저 작성합니다.


<div style={{ height: "6px" }} />

#### Step 2. 자동화를 시작할 리스트를 준비합니다
작성한 이메일은 시작 리스트에 연결되어야 발송됩니다. 시작 리스트는 다음 두 가지 방법으로 준비할 수 있습니다. 
1. **직접 추가하기**: 기존 연락처 리스트를 선택하여 리스트에 추가하는 방법  
2. **폼 응답 연결하기**: 폼으로 들어온 응답이 자동으로 리스트에 추가되도록 설정하는 방법

<div style={{ height: "6px" }} />

#### Step 3. 자동화 탭에서 시작 리스트를 연결합니다

상단의 `자동화` 탭을 클릭하면, 하나의 이메일을 중심으로 앞과 뒤에 연결될 리스트를 설정할 수 있습니다.

<img

  style={{

/>
`+ 시작 리스트 추가` 버튼을 클릭하여 시작 리스트를 추가하면 연결된 이메일이 자동으로 발송됩니다.

시작 리스트를 추가할 때는 다음 항목을 함께 설정할 수 있습니다.

**(1) 리스트 선택**  : 자동화를 시작할 기준 리스트를 선택합니다. 선택한 리스트에 연락처가 추가되면 자동화가 실행됩니다.

**(2) 필터 추가**  : 필요한 경우, 선택한 리스트 중 특정 조건을 만족한 항목에 대해서만 자동화를 실행하도록 설정할 수 있습니다. 예를 들어,  연락처 이름에 특정 문자가 포함된 경우나 특정 필드를 기준으로 필터링한 경우와 같이 자동화 실행 대상을 세분화할 수 있습니다.

**(3) 발송 시점 (지연)**  : 리스트에 항목이 추가된 이후 이메일을 언제 발송할지를 설정합니다.   
- 즉시 발송 (immediately)    
- 일정 시간 후 발송 (시간 또는 일 단위)


<div style={{ height: "6px" }} />

#### Step 4. 대상 리스트를 추가하고, 반응 조건을 설정합니다

이메일에 대한 리드의 반응에 따라 이동할 대상 리스트를 설정할 수 있습니다. 리드가 특정 반응과 조건을 만족했을 때만 이동하는 리스트입니다.

`+ 대상 리스트 추가` 버튼을 클릭하여 대상 리스트를 추가합니다.
<img

  style={{

대상 리스트를 추가할 때는 다음 항목들을 함께 설정합니다.

**(1) 리스트 선택** : 리드가 설정된 조건을 만족했을 시 이동할 리스트를 선택합니다.

**(2) 트리거** : 어떤 이메일 반응을 기준으로 대상 리스트로 이동할지를 선택합니다.  
- 이메일 열람/클릭 시  
- 수신 거부 시

**(3) 조건 추가**  : 필요한 경우, 선택한 트리거 중에서도 특정 기준을 만족한 경우에만 대상 리스트로 이동하도록 조건을 추가할 수 있습니다.
- 이메일 열람 횟수가 특정 횟수 이상  
- 이메일 총 클릭 수가 특정 횟수 이상  
- 특정 링크를 특정 횟수 이상 클릭한 경우  

<div style={{ marginBottom: "24px" }}>
  대상 리스트를 활용하면 같은 이메일을 받았더라도 리드의 반응에 따라 서로 다른 다음 흐름으로 자연스럽게 분기할 수 있습니다.
</div>


<div style={{ height: "8px" }} />

#### Step 5. 대상 리스트는 다음 자동화 세트의 시작점이 됩니다

조건을 만족한 리드는 설정된 대상 리스트로 이동하며, 이 리스트는 다시 다음 자동화의 시작 리스트로 사용될 수 있습니다.

## Emma 자동화 전체 흐름의 시각적 구조
왼쪽 사이드바에서 `자동화` 페이지를 클릭하면, Emma에서 설정된 모든 자동화 흐름을 하나의 화면에서 확인할 수 있습니다.
각 리드가 어떤 리스트에서 시작해 어떤 이메일을 거쳐, 어떤 조건에 따라 다음 리스트로 이동하는지 시각적으로 표현되며, 전체 자동화 구조를 흐름 단위로 관리할 수 있습니다.
