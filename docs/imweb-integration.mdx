## Overview

아임웹 사용자분들이 스프레드를 사용해서 이메일을 보내기 위해 먼저 아임웹-스프레드 연동이 필요합니다.

## 1단계: 스프레드 연동 시작하기

1. 앱스토어의 '사이트에 추가' 버튼을 누릅니다.
2. 스프레드 계정 설정 화면에서 비밀번호를 설정합니다. 계정의 이메일 주소는 사이트의 '대표메일' 입니다.
    a. 대표메일은 아임웹 관리자 페이지 > 왼쪽 사이드바 메뉴 [환경설정 > 일반 > 상점/사이트 정보 설정] 메뉴에서 확인할 수 있습니다.
    b. 비밀번호는 12자리 이상으로 설정해주세요.
3. 기존 스프레드/릴레잇에서 사용하던 계정이 있는 경우, ‘사이트에 추가’ 를 누르면 이동하는 스프레드 로그인 화면에서 로그인해주시면 아임웹 사이트와 연동됩니다.
4. 비밀번호 설정을 마치면 스프레드 관리자페이지로 이동합니다.

![imweb-spread1](/images/imweb1.png)

## 2단계: 스프레드 워크스페이스 설정하기

1. 언어 변경이 필요한 경우, 스프레드 관리자 페이지 우측 상단의 지구본 아이콘을 클릭해 '한국어' 로 변경합니다.
    ![imweb-spread2](/images/imweb2.png)
2. 워크스페이스 설정하기
    ![imweb-spread3](/images/imweb3.png)
    a. 워크스페이스 이름은 아임웹 웹사이트의 ‘내 사이트 이름’ 과 동일하게 설정됩니다.
    b. 워크스페이스의 이름과 시간대를 설정할 수 있습니다. 시간대는 이메일 예약 발송 시 활용되니, 시간대를 '(UTC+09:00) Asia/Seoul' 로 변경해주세요.
3. 사용자관리: 추가 비용 없이, 이메일 발송을 함께 관리할 팀원을 초대하실 수 있습니다.
4. 요금제
    a. 회원(구독자) 1,000 명 미만은 무료로 사용하실 수 있습니다. 이메일을 보내는 구독자가 많아지면 요금제를 업그레이드 해주세요.
    b. 회원(구독자) 이메일이 더 이상 유효하지 않거나, 바운스 등으로 확인되는 경우 회원(구독자) 수 계산에 포함되지 않습니다.
    c. 자동 이메일 캠페인 (미리 작성된 이메일을 회원이 추가되면 발송하는 기능)은 Spread Pro 2K 요금제부터 사용하실 수 있습니다.

## 3단계: 연락처 동기화 및 업로드

1. 아임웹과 연동이 완료된 스프레드 워크스페이스에는 아임웹 웹사이트의 사용자 정보와 스프레드의 고객 연락처가 자동으로 동기화됩니다.
2. 현재 아임웹 내 사이트에 가입/추가된 회원 외, 고객 연락처를 업로드하고 싶으시다면 아임웹 웹사이트 설정페이지에서 업로드 해주세요.
    ![imweb-spread4](/images/imweb4.png)
    a. 아임웹 관리자 페이지 > 왼쪽 사이드바 메뉴 [사용자 관리 > 사용자 목록] > 우측 상단 '대량 사용자 추가' 에서 하실 수 있습니다.
    b. 연락처를 업로드 한 뒤에는 스프레드 앱 우측 상단의 '지금 동기화' 버튼을 눌러 연락처를 동기화해주세요.
    ![imweb-sync](/images/imweb-sync.png)

## 4단계: 스프레드 앱 실행하기

1. 스프레드 관리자 메뉴 우측 상단의 '스프레드 실행하기' 버튼을 눌러 스프레드 앱을 실행합니다. 한 번 더 로그인해주세요. (이메일: 아임웹 웹사이트 대표메일, 비밀번호: 최초 연동 시 설정한 비밀번호)
2. 스프레드 앱 우측 상단에 내 사이트 이름과 아임웹 로고가 보인다면 정상적으로 연동된 것입니다.
    ![imweb-spread5](/images/imweb5.png)

## 5단계: 웹사이트 회원(구독자) 연락처 확인

1. 좌측 사이드바의 '전체 연락처' 메뉴에서 이메일을 보낼 수 있는 회원(구독자)의 이메일 주소와 정보를 확인할 수 있습니다.
    ![imweb-spread6](/images/imweb6.png)
2. 좌측 사이드바의 '리스트' 메뉴에서 이메일 수신자 그룹으로 설정할 수 있는 '리스트'를 확인할 수 있습니다.
3. 각 '리스트' 안에는 아임웹 웹사이트에서 '그룹' 으로 분류한 회원(구독자) 정보를 볼 수 있습니다.

## 6단계: 새로운 리스트 생성하기

1. 좌측 사이드바의 '리스트' 메뉴 접근 > 상단의 '+리스트' 버튼을 눌러 새로운 리스트를 추가할 수 있습니다.
2. 새로 생성한 리스트에 들어가면, 상단의 '+연락처 > 모든 연락처에서 추가' 버튼을 눌러 전체 연락처에서 해당 리스트에 추가할 연락처를 선택할 수 있습니다.
3. 좌측 사이드바 '전체 연락처' 메뉴에서도 필터 기능을 활용해 특정 정보를 가진 회원(구독자)의 정보를 리스트에 추가할 수 있습니다.
    a. 필터 조건을 추가한 뒤, 목록 좌측의 체크박스를 통해 대량으로 선택 후, 하단에 있는 '리스트에 추가' 버튼을 눌러 추가할 리스트를 선택합니다.
    ![imweb-spread7](/images/imweb7.png)

## 7단계: 아임웹에서 구매한 도메인 주소 연결하기

### 커스텀 도메인 주소가 없는 경우

1. 아직 커스텀 도메인 주소(예: relate.so)가 없는 경우, 스프레드에서 이메일은 'spreadsend.com' 도메인 주소로 발송하실 수 있습니다.
2. 커스텀 도메인이 없는 경우, 발송하는 이메일 주소는 워크스페이스의 이름입니다. 발송 주소를 변경하고 싶으신 경우 워크스페이스 이름을 변경해주세요.
3. 이 경우 답장 받을 이메일인 Reply to 를 꼭 설정해주세요.
    ![imweb-spread8](/images/imweb8.png)

### 커스텀 도메인 주소(예: spread.so)가 있는 경우

1. 내 웹사이트에서 사용하는 도메인 주소로 발송하고자 하는 경우, 좌측 상단 [워크스페이스 로고 클릭 > 설정 > 도메인] 으로 이동해 도메인 정보를 추가할 수 있습니다.
    ![imweb-spread9](/images/imweb9.png)
    a. 내가 보내는 메일이 스팸으로 분류되지 않고 원활하게 전달될 수 있도록 가급적 커스텀 도메인을 활용하실 것을 권장합니다.
    b. 도메인 추가 버튼을 눌러 이메일 주소에 사용할 도메인 주소를 입력해주세요. (예: abc@spread.com -> spread.com 그대로 입력)
    c. 아래 이미지와 같은 정보가 확인되면, 창을 닫지 말고 다음 단계로 진행해주세요.
        ![imweb-spread10](/images/imweb10.png)
2. 아임웹 웹사이트에서 구매한 커스텀 도메인 주소 설정하기
    a. [아임웹 웹사이트 관리자 페이지 > 환경설정 > 도메인] 에 접속하여 아임웹에서 구매한 도메인 하단의 '네임서버 설정' 으로 이동하세요.
        ![imweb-spread11](/images/imweb11.png)
        ![imweb-spread12](/images/imweb12.png)
    b. 아래 네임서버 설정 팝업 하단의 '레코드 추가' 버튼을 눌러 Sending Records 와 Tracking Records 에 있는 정보를 입력합니다. 모두 입력하신 후에는 하단 '저장' 버튼을 통해 저장해주세요.
    c. 다시 스프레드의 도메인 설정 페이지로 이동해 하단의 '레코드 확인' 버튼을 눌러주세요.
3. 정상적으로 인증이 되면 아래와 같이 도메인 레코드가 인증되었다는 화면을 확인하실 수 있습니다.
    ![imweb-spread13](/images/imweb13.png)
    a. SPF&DMARC 정보는 도메인 주소에 따라 인증에 30분 정도 시간이 소요될 수 있습니다. 시간이 오래 지나도 인증되지 않는다면 스프레드 고객센터 ([we@relate.so](we@relate.so) 이메일, 앱 우측 하단 채팅)
4. Cloudflare, Namecheap 등 타 사이트에서 도메인을 구매하신 경우는 [이 문서](https://docs.spread.so/domain)를 참고해주세요.

회원 정보를 연동하고 커스텀 도메인까지 연결했다면, 스프레드로 이메일을 보낼 준비를 모두 마치셨습니다!

이후 이메일 캠페인 작성, 자동 이메일 캠페인 운영 방법 등은 스프레드 가이드문서의 [이 항목](https://docs.spread.so/send-email)에서부터 시작해보세요 💌
